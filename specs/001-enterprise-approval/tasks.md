- [x] T000-02 确认 Spring Boot 版本。
  - 来源：`AGENTS.md` 第 3.1 章，`plan.md` 第 3 章。
  - 决策：采用 Spring Boot 2.7.x + Java 1.8。
  - 范围：所有 Java 代码必须兼容 Java 1.8，不使用 Java 9+ 语法和 API。
  - 验收：已在 `AGENTS.md` 和 `plan.md` 中同步最终规则。
